## How to get added

**For Patreon patrons:** after your first payment clears, DM me on Discord (or send a direct message through Patreon) with:
- The **display name** you'd like shown (real name, handle, or studio name).
- An **avatar image** (optional) — see the avatar format notes below.
- A **link** (optional) — personal site, GitHub, studio, or project page. Sponsors may include a one-line tagline.
- Whether you'd like your **tier** acknowledged publicly or listed anonymously (e.g. "Anonymous Supporter").

I'll add you to the appropriate tier section within a few days.

**For Ko-fi supporters:** include a note with your tip if you'd like to be listed. Anonymous tips won't be listed by default — DM me on Discord if you tipped anonymously and want your name added.

**Privacy:** you can be listed under any name you choose (handle, real name, or studio). Avatars and links are entirely optional. You can change or remove your listing at any time by DMing me on Discord.

### Avatar format (optional)

If you'd like your avatar shown next to your name:

- **Format:** PNG, JPG, or WebP. Square works best.
- **Size:** 128×128 px minimum, 256×256 px preferred. I'll downscale for display.
- **Filename:** please name it `yourname.png` (lowercase, no spaces) — e.g. `jane-doe.png` or `stormbyte-studio.png`.
- **Where it's stored:** `media/supporters/` in this repository. The image is committed to the repo, so it's public — only send an image you're comfortable having permanently visible.
- **Alternatively:** if you'd rather not send an image, I can use your GitHub avatar (just include your GitHub username) or a default Knot-themed placeholder.

The listing format in Markdown looks like:

```markdown
![Jane Doe](media/supporters/jane-doe.png) **[Jane Doe](https://janedoe.dev)** — _Interactive fiction author_
```

…which renders as an avatar next to your name and link. Sponsors get the same treatment in the **Special Thanks** section at the top.

## Notes

- Listings are curated manually — I don't have automated sync with Patreon or Ko-fi yet.
- If your tier changes (upgrade or downgrade), I'll move your listing to the matching section. Top-billing for Sponsors is reserved for active Sponsor-tier patrons.
- This file is checked into the repo, so additions appear in the project's commit history. That's intentional — it makes the thank-you permanent.
- If you'd prefer **not** to be listed at all, just let me know and I'll keep your support anonymous.
